Sistema de Apoyos usando Pilas y Colas.
Usa los botones para navegar entre las opciones.
Descripción:
Este sistema web permite gestionar la entrega de apoyos mediante dos estructuras de datos:
  • Cola: Para la entrega de tarjetas de apoyo. Simula personas esperando su turno. Puedes agregar personas a la cola y atenderlas en orden.
  • Pila: Para el historial de apoyos recibidos. Permite ver los apoyos y "deshacer" el último recibido.
El sistema cuenta con botones para agregar, atender y deshacer apoyos, y una interfaz visual clara y moderna.
